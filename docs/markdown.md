# Markdown em 5min

## Cabeçalhos
```
# Cabeçalho H1
## Cabeçalho H2
### Cabeçalho H3
#### Cabeçalho H4
##### Cabeçalho H5
###### Cabeçalho H6
```

## Formatação de texto
```
**texto em negrito**
*texto em itálico*
***negrito e itálico***
~~tachado~~
`código em linha`
```

## Links e imagens
```
[Texto do link](https://exemplo.com)
[Link com título](https://exemplo.com "Título ao passar o mouse")
![Texto alternativo](imagem.jpg)
![Imagem com título](imagem.jpg "Título da imagem")
```

## Listas
```
Não ordenadas:
- Item 1
- Item 2
  - Item aninhado

Ordenadas:
1. Primeiro item
2. Segundo item
3. Terceiro item
```

## Citações
```
> Esta é uma citação
> Múltiplas linhas
>> Citação aninhada
```

## Blocos de código
````
```javascript
function ola() {
  console.log("Olá, mundo!");

````

## Tabelas
```
| Cabeçalho 1 | Cabeçalho 2 | Cabeçalho 3 |
|-------------|-------------|-------------|
| Linha 1     | Dados       | Dados       |
| Linha 2     | Dados       | Dados       |
```

## Linha horizontal
```
---
ou
***
ou
___
```

## Listas de tarefas
```
- [x] Tarefa concluída
- [ ] Tarefa incompleta
- [ ] Outra tarefa
```

## Escapando caracteres
```
Use barra invertida para escapar: \* \_ \# \`
```

## Quebras de linha
```
Termine uma linha com dois espaços  
para criar uma quebra de linha.

Ou use uma linha em branco para um novo parágrafo.
```
