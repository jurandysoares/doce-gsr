# doce-gsr

Documentação para disciplina Gerência e Segurança de Redes

## Instalação das ferramentas

1. `curl -LsSf https://astral.sh/uv/install.sh | sh`
1. `uv tool install mkdocs`
1. `mkdocs`
1. `mkdocs new .`
1. `mkdocs serve`
1. `mkdocs gh-deploy`
1. `git add .`
1. `git commit -m "MENSAGEM"`
1. `git push`
